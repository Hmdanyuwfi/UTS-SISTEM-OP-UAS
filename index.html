Tutorial Instalasi Linux Debian di Oracle VirtualBox

Panduan lengkap untuk menginstal sistem operasi Linux Debian menggunakan Oracle VirtualBox

Persiapan Sebelum Instalasi

  • Oracle VirtualBox terinstal di komputer Anda
  • File ISO Linux Debian (download dari situs resmi debian.org)
  • Minimal 2GB RAM untuk virtual machine
  • Minimal 20GB ruang penyimpanan kosong
  • Koneksi internet untuk update dan instalasi paket tambahan

Langkah 1: Membuat Virtual Machine Baru

  1. Buka Oracle VirtualBox
  2. Klik tombol "New" atau "Baru" untuk membuat VM baru
  3. Masukkan nama untuk VM (contoh: "Debian Linux")
  4. Pilih Type: "Linux"
  5. Pilih Version: "Debian (64-bit)"
  6. Klik "Next" untuk melanjutkan

Langkah 2: Konfigurasi Memory (RAM)

  1. Atur jumlah RAM yang akan dialokasikan (minimal 1024MB, disarankan 2048MB)
  2. Pastikan tidak mengalokasikan lebih dari setengah RAM fisik komputer
  3. Klik "Next" untuk melanjutkan

Langkah 3: Membuat Hard Disk Virtual

  1. Pilih "Create a virtual hard disk now"
  2. Klik "Create"
  3. Pilih tipe file hard disk: "VDI (VirtualBox Disk Image)"
  4. Pilih "Dynamically allocated" untuk efisiensi ruang
  5. Atur ukuran hard disk (minimal 20GB, disarankan 30GB)
  6. Klik "Create" untuk menyelesaikan

Langkah 4: Konfigurasi Tambahan VM

  1. Pilih VM yang telah dibuat, klik "Settings"
  2. Di tab "System" > "Processor", atur jumlah CPU cores (1-2 cores)
  3. Di tab "Display", aktifkan "Enable 3D Acceleration"
  4. Di tab "Storage", klik ikon CD, pilih "Choose a disk file"
  5. Pilih file ISO Debian yang telah didownload
  6. Klik "OK" untuk menyimpan pengaturan

Langkah 5: Memulai Instalasi Debian

  1. Klik "Start" untuk menjalankan VM
  2. VM akan boot dari ISO Debian
  3. Pilih "Install" atau "Graphical Install" dari menu boot
  4. Pilih bahasa instalasi (English atau Indonesian)
  5. Pilih lokasi geografis (Indonesia)
  6. Konfigurasi keyboard layout

Langkah 6: Konfigurasi Jaringan dan Pengguna

  1. Masukkan hostname untuk sistem (contoh: "debian-vm")
  2. Masukkan domain name (bisa dikosongkan)
  3. Buat password untuk root user
  4. Buat user account baru dengan nama lengkap
  5. Masukkan username dan password untuk user tersebut

Langkah 7: Partisi Hard Disk

  1. Pilih "Guided - use entire disk" untuk partisi otomatis
  2. Pilih hard disk virtual yang telah dibuat
  3. Pilih "All files in one partition" untuk kemudahan
  4. Konfirmasi perubahan partisi dengan memilih "Yes"

Langkah 8: Instalasi Sistem Base

  1. Tunggu proses instalasi sistem base (5-15 menit)
  2. Konfigurasi package manager (pilih mirror terdekat)
  3. Pilih apakah akan menggunakan proxy (biasanya "No")
  4. Pilih paket software yang akan diinstal:
    • Debian desktop environment
    • GNOME (atau desktop environment pilihan)
    • Standard system utilities

Langkah 9: Instalasi GRUB Bootloader

  1. Pilih "Yes" untuk menginstal GRUB bootloader
  2. Pilih hard disk untuk instalasi GRUB (/dev/sda)
  3. Tunggu proses instalasi selesai

Langkah 10: Menyelesaikan Instalasi

  1. Setelah instalasi selesai, klik "Continue"
  2. VM akan restart otomatis
  3. Lepas ISO dari virtual drive (Devices > Optical Drives > Remove disk)
  4. Login menggunakan user account yang telah dibuat
  5. Sistem Debian siap digunakan!

Tips Tambahan

  • Install Guest Additions: Untuk performa yang lebih baik, install VirtualBox Guest Additions melalui menu Devices
  • Update Sistem: Jalankan sudo apt update && sudo apt upgrade untuk update sistem
  • Snapshot: Buat snapshot setelah instalasi selesai untuk backup
  • Shared Folder: Konfigurasi shared folder untuk berbagi file dengan host OS
  • Network: Gunakan NAT untuk akses internet atau Bridged untuk akses jaringan lokal
